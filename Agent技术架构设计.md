## 1. 项目概览

### 1.1 核心定位

基于开源生态构建的**生物医学实验智能体系统**，以大语言模型为核心推理引擎，通过标准化工具协议（MCP）集成现有CV能力，实现自然语言驱动的自主实验分析。

**关键承诺**：70%代码复用成熟开源项目，30%原生开发聚焦科学约束与质量感知。

### 1.2 业务目标

| 目标             | 指标     | 验收标准                      |
| ---------------- | -------- | ----------------------------- |
| 降低实验分析门槛 | 操作步骤 | 从10+配置步骤→1句自然语言     |
| 提升分析可靠性   | 失败率   | 自动修复率>80%，人工介入<20%  |
| 加速知识沉淀     | 经验复用 | 新实验类型配置时间从2天→2小时 |
| 保障科学严谨     | 可复现性 | 完整实验谱系记录，支持审计    |

### 1.3 约束条件

- **预算**：软件成本<$2,000/月（API+基础设施）
- **时间**：8周MVP，12周生产就绪
- **人力**：2名工程师（1名CV背景，1名Agent背景）
- **合规**：实验数据可本地处理，支持私有化部署

## 3. 技术选型详单

### 3.1 开源复用组件（70%）

| 组件           | 选型           | 版本         | 用途           | 替代方案                    |
| -------------- | -------------- | ------------ | -------------- | --------------------------- |
| **Agent框架**  | LangGraph      | ^0.2.0       | 工作流编排     | AutoGen（微软，复杂度高）   |
| **LLM路由**    | LiteLLM        | ^1.0.0       | 多模型统一接入 | 自研（不推荐，LiteLLM成熟） |
| **记忆层**     | Mem0           | ^1.0.0       | 分层记忆管理   | 自研（Mem0已优化Agent场景） |
| **向量数据库** | ChromaDB       | ^0.5.0       | 记忆与文献检索 | Milvus（大规模时考虑）      |
| **工具协议**   | MCP SDK        | ^1.0.0       | 工具标准化     | 自定义（失去生态兼容）      |
| **API框架**    | FastAPI        | ^0.110.0     | REST + SSE     | Flask（性能不足）           |
| **任务队列**   | Celery + Redis | ^5.3.0       | 异步执行       | Temporal（长时任务时升级）  |
| **数据库**     | PostgreSQL 15  | 15.x         | 关系数据       | MySQL（JSON支持弱）         |
| **对象存储**   | MinIO          | RELEASE.2024 | 视频/结果存储  | AWS S3（云端部署时）        |
| **UI框架**     | Gradio         | ^4.0.0       | 快速Web界面    | Streamlit（灵活性差）       |
| **本地LLM**    | Ollama         | ^0.3.0       | 本地模型服务   | vLLM（高并发时）            |

### 3.2 原生开发组件（30%）

| 组件             | 技术               | 代码量估算 | 关键挑战           |
| ---------------- | ------------------ | ---------- | ------------------ |
| **Protocol本体** | Pydantic + YAML    | 500行      | 领域知识编码       |
| **约束验证器**   | Pydantic Validator | 300行      | 硬约束与LLM协作    |
| **CV质量感知**   | 你的经验规则       | 400行      | 概率性输出处理     |

| **确认点UI**     | Gradio组件         | 400行      | 科学家友好设计     |
